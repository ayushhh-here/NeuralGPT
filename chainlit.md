# 🧠 NeuralGPT

Welcome! You're chatting with **NeuralGPT** — a GPT-style transformer built from scratch in pure PyTorch and instruction fine-tuned on top of GPT-2 (355M) weights.

No HuggingFace Transformers, no pre-built attention modules — every layer (token embeddings, positional encodings, multi-head causal self-attention, feed-forward blocks, layer norm) was implemented by hand.

Type a prompt below and the model will generate a response token by token.

Source & write-up: [github.com/ayushhh-here/NeuralGPT](https://github.com/ayushhh-here/NeuralGPT)
